# Supernova Design System Guide

## 세련된 미니멀리즘 (Refined Minimalism)

### 핵심 철학

1. **본질에 집중**: 불필요한 요소는 제거하되, 사용자 경험을 해치지 않는다.
2. **여백의 조화**: 콘텐츠와 여백의 균형을 맞춘다 (60:40 비율).
3. **명확한 계층**: 각 화면의 주요 메시지와 보조 정보를 분명히 구분한다.
4. **숨 쉴 수 있는 공간**: 요소 사이에 충분한 거리를 두되, 단조롭지 않게.

## 색상 팔레트

### 기본 (모노크롬 + 포인트)
- **배경**: 순수한 흰색 `#FFFFFF` 또는 미세한 회색 `#FAFAFA`
- **텍스트**: 진한 검정 `#0A0A0A` (primary), 중간 회색 `#6B7280` (secondary), 연한 회색 `#9CA3AF` (tertiary)
- **강조**: 검정 `#000000` (버튼, 중요 액션)
- **포인트**: 선택적으로 브랜드 컬러 1가지 (예: `#6366F1` 인디고) - 버튼이나 강조점에만 사용
- **경계선**: 아주 연한 `#F3F4F6` 또는 없음
- **배경 변주**: 섹션 구분을 위해 `#F9FAFB` 연한 회색 사용 가능

### 허용되는 색상 사용
- ✅ 브랜드 포인트 컬러 1가지 (선택적)
- ✅ 은은한 그라디언트 (회색 계열)
- ✅ 호버 상태의 색상 변화

### 금지 사항
- ❌ 3가지 이상의 강렬한 컬러
- ❌ 네온/형광 색상
- ❌ 과도한 그라디언트

## 타이포그래피

### 폰트
- **한글**: **Pretendard** (필수 사용)
- **영문/숫자**: Inter, SF Pro
- **폰트 로드**: 
```html
<link rel="stylesheet" as="style" crossorigin href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.min.css" />
```

### 크기 시스템
- **대형 제목**: 48-64px (text-5xl ~ text-6xl)
- **중형 제목**: 32-40px (text-3xl ~ text-4xl)
- **소형 제목**: 24-28px (text-2xl ~ text-3xl)
- **본문**: 16-18px (text-base ~ text-lg)
- **보조 텍스트**: 14px (text-sm)
- **자간**: 제목은 타이트하게 `-0.02em`, 본문은 기본 또는 `0.01em`

### 한글 텍스트 가이드
- 제목은 짧고 임팩트 있게 (10-15자)
- 부제목은 1-2줄로 핵심 가치 전달
- 버튼 텍스트는 3-5자 (시작하기, 더 알아보기, 무료 체험)
- 본문은 간결하되 이해하기 쉽게 (20-30자 이내)

## 여백 시스템

### 섹션 간격
- 섹션 사이: `py-24` ~ `py-32` (적당히 넓게)
- 내부 패딩: `px-8` ~ `px-12`
- 요소 간격: `gap-8` ~ `gap-12`
- 컨텐츠 그룹 간격: `gap-6`

### 컨테이너
- 최대 너비: `max-w-6xl` 또는 `max-w-7xl` (적당히 넓게)
- 중앙 정렬: `mx-auto`
- 좌우 패딩: `px-4` ~ `px-8`

## 컴포넌트 스타일

### 버튼
```
Primary: bg-black text-white px-8 py-4 rounded-full font-medium
         hover:bg-gray-900 transition-all duration-300

Secondary: bg-transparent text-black border border-black px-8 py-4 rounded-full
           hover:bg-black hover:text-white transition-all duration-300

Text: text-gray-600 hover:text-black underline-offset-4 hover:underline
```

### 카드
```
bg-white border border-gray-100 rounded-3xl p-10
또는 배경 없이 텍스트만
```

### 네비게이션
```
- 로고 (좌측) + 텍스트 링크 (우측)
- 배경: 투명 또는 흰색
- 높이: 최소화 (h-16)
- 경계선: 없음 또는 매우 연함
```

## 레이아웃 패턴

### 히어로 섹션
```
- 중앙 정렬, 단일 메시지
- 대형 제목 (text-5xl ~ text-7xl)
- 짧은 부제목 (text-xl, text-gray-600)
- 버튼 1-2개
- 배경: 순수 흰색
```

### 피처 섹션
```
- 3컬럼 또는 2컬럼
- 아이콘 최소화 (선택적)
- 제목 + 한 줄 설명
- 충분한 간격 (gap-12)
```

### 푸터
```
- 매우 심플하게
- 로고 + 핵심 링크만
- 저작권 텍스트
```

## 애니메이션

### 허용
- 부드러운 호버 전환 (`transition-all duration-300`)
- 미세한 스케일 변화 (`hover:scale-[1.02]`)
- 페이드 인/아웃

### 금지
- ❌ 바운스 효과
- ❌ 복잡한 모션
- ❌ 과한 전환 효과

---

## 아이콘 가이드

### 스타일
- Lucide Icons 사용
- 선 두께: 1.5px (얇게)
- 크기: 20-24px
- 색상: 현재 텍스트 색상과 동일

### 사용 원칙
- 필수적인 경우에만 사용
- 텍스트로 충분하면 아이콘 제거
- 장식용 아이콘 금지

---

## 이미지 가이드

### 플레이스홀더
- Unsplash 고품질 이미지 사용
- 미니멀하고 깔끔한 사진 선택
- 모노톤 또는 저채도 이미지 선호

### 처리
- `rounded-2xl` 또는 `rounded-3xl`
- 그림자 없음 또는 매우 연함
- `object-cover` 비율 유지
