# 랜딩페이지 필수 구성 요소

> 모든 랜딩페이지는 아래 요소들을 기본으로 포함하여 구성합니다.
> 순서는 유연하게 조정 가능하며, 프로젝트 특성에 맞게 섹션을 선택적으로 적용합니다.

---

## 1️⃣ **헤더 & 상단 히어로 섹션**

### 포함 요소
- 🔥 **긴급성 문구** (예: "48시간 한정", "선착순 100명")
- ⏰ **타이머 / 카운트다운** (시각적 긴박감)
- ✨ **핵심 혜택 한 문장** (30자 이내, 명확한 가치 제안)
- ⭐ **짧은 사회적 증거** (별점 4.8/5.0, 1,234명 사용 중)
- 🎯 **메인 CTA 버튼** (무료 시작하기, 지금 신청하기)
- 📢 **메인 헤드라인** (48-64px, 굵게, 임팩트 있게)
- 📝 **서브헤드라인** (18-20px, 부가 설명)

### 디자인 패턴
```
[상단 배너: 긴급성 문구]
[네비게이션: 로고 + 메뉴]
[히어로 영역]
  - 좌측: 헤드라인 + 서브헤드라인 + CTA
  - 우측: 제품 이미지 또는 영상
  - 하단: 신뢰 배지 (별점, 사용자 수)
```

## 2️⃣ **문제 공감(Pain) 섹션**

### 포함 요소
- 💭 **"이런 고민 있으시죠?"** 패턴
- 📋 **구체적 상황 3~5가지** (체크리스트 형태)
- ❌ **기존 방법 실패 이유** (왜 지금까지의 방법이 안 통했는지)
- 🔍 **문제의 근본 원인** (통찰력 있는 진단)

### 디자인 패턴
```
[섹션 제목: "이런 어려움을 겪고 계신가요?"]
[체크박스 리스트]
  ☑️ 문제 상황 1
  ☑️ 문제 상황 2
  ☑️ 문제 상황 3
[강조 박스: "사실, 이건 당신 잘못이 아닙니다"]
```

## 3️⃣ **문제 정의 + 해결책 제안**

### 포함 요소
- 📉 **현재 상태(AS-IS)** (Before)
- 📈 **해결 후 상태(TO-BE)** (After)
- 🚫 **왜 기존 방식이 실패하는지**
- 💡 **핵심 솔루션 한 문장** (명확한 제안)

### 디자인 패턴
```
[2컬럼 대비 레이아웃]
  좌측: AS-IS (회색 배경, 😞)
  우측: TO-BE (밝은 배경, 😊)
[화살표: → ]
[해결책 요약 박스]
```

---

## 4️⃣ **스토리텔링 섹션**

### 포함 요소
- 📖 **Before**: 과거의 고통/실패 경험
- 💡 **Turning Point**: 해결책을 발견한 결정적 순간
- ✨ **After**: 현재의 변화와 성과
- 💪 **"당신도 가능합니다"** 메시지

### 디자인 패턴
```
[타임라인 형태 또는 3단 스토리]
[이미지 + 텍스트 조합]
[감정선이 드러나는 문구]
```

---

## 5️⃣ **솔루션 상세 설명**

### 포함 요소
- 🎯 **기능/구성/모듈 설명** (3-5개)
- 📊 **단계별 프로세스** (STEP 1 → 2 → 3)
- 🖼️ **실제 화면** (스크린샷, 목업)
- ⚡ **차별화 포인트** (경쟁사 대비 우위)

### 디자인 패턴
```
[3컬럼 피처 그리드]
  [아이콘 + 제목 + 설명]
[또는 단계별 프로세스]
  STEP 1 → STEP 2 → STEP 3
```

---

## 6️⃣ **사회적 증거 섹션**

### 포함 요소
- 📊 **숫자로 보는 성과** (전환율 +320%, 매출 2배)
- 💬 **후기 캡처** (실제 사용자 리뷰 3-5개)
- 🎥 **영상 후기** (선택적)
- 📱 **실시간 증거** (카톡방 스크린샷, SNS 멘션)

### 디자인 패턴
```
[숫자 강조 섹션]
  1,234명 사용  |  4.8/5.0 평점  |  98% 만족도
[후기 카드 그리드]
  [프로필 이미지 + 이름 + 별점 + 후기 내용]
```

---

## 7️⃣ **권위 섹션**

### 포함 요소
- 👤 **창업자/강사 경력** (전문성 강조)
- 📰 **미디어 노출** (언론사 로고)
- 🏢 **고객사/파트너 로고** (신뢰도)
- 👨‍🏫 **전문가 추천사** (인플루언서, 업계 전문가)

### 디자인 패턴
```
[로고 월 - 회색톤 필터]
  [언론사/파트너 로고 4-8개]
[프로필 카드]
  [사진 + 이름 + 경력 + 한줄 소개]
```

---

## 8️⃣ **사용성 강조 섹션**

### 포함 요소
- ✅ **"정말 쉽습니다"** 메시지
- 📋 **단계별 진행 방법** (1-2-3 스텝)
- 👶 **초보자도 가능한 이유**
- 💁 **지원 시스템** (가이드/튜토리얼/CS)

### 디자인 패턴
```
[3스텝 프로세스]
  1. 가입하기 (30초)
  2. 설정하기 (2분)
  3. 시작하기 (바로)
[각 스텝마다 아이콘 + 설명]
```

---

## 9️⃣ **혜택 종합 + 보너스**

### 포함 요소
- 🎁 **핵심 혜택 리스트** (5-7개)
- 💎 **보너스 항목** (금액 환산: 총 가치 ₩500,000)
- ⚡ **한정 혜택** (얼리버드 30% 할인)
- 📦 **시각적 혜택 나열** (체크리스트 형태)

### 디자인 패턴
```
[혜택 박스]
  ✓ 혜택 1 (₩100,000 상당)
  ✓ 혜택 2 (₩150,000 상당)
  ✓ 보너스 1 (₩50,000 상당)
[총 가치 강조: ₩500,000 → 오늘만 ₩99,000]
```

## 🔟 **리스크 제거 섹션**

### 포함 요소
- 💰 **환불 정책** (30일 무조건 환불)
- 🛡️ **보장/보증 문구** (100% 만족 보장)
- ❓ **FAQ** (주요 우려사항 5-7개)
- 🔒 **인증/보안 배지** (SSL, PCI DSS 등)

### 디자인 패턴
```
[보장 배지 섹션]
  [아이콘 + 30일 환불 보장]
[FAQ 아코디언]
  Q: 질문 1
  A: 답변 1
```

## 1️⃣1️⃣ **Before / After 최종 대비**

### 포함 요소
- 😰 **선택하지 않았을 때의 1년 후** (Bad Ending)
- 😊 **선택했을 때의 1년 후** (Happy Ending)
- 📊 **감정 대비** (불안 vs 자신감)
- ⏰ **시간 대비** (6개월 vs 1주일)

### 디자인 패턴
```
[2컬럼 대비]
  좌측 (어두운 배경):
    ❌ 1년 후에도 같은 문제
    ❌ 시간과 돈만 낭비
  우측 (밝은 배경):
    ✅ 1년 후 목표 달성
    ✅ 새로운 기회 확보
```

## 1️⃣2️⃣ **타겟 필터링**

### 포함 요소
- ✅ **이런 분에게 추천** (3-5개 프로필)
- ❌ **이런 분에겐 비추천** (정직한 가이드)
- 🎯 **현실적 기대치 안내**

### 디자인 패턴
```
[2컬럼 레이아웃]
  좌측: 
    ✅ 추천 대상 1
    ✅ 추천 대상 2
  우측:
    ❌ 비추천 대상 1
    ❌ 비추천 대상 2
```

## 1️⃣3️⃣ **최종 CTA (Call-to-Action)**

### 포함 요소
- ⚡ **긴급성 재강조** ("오늘 자정 마감")
- 💰 **핵심 혜택 리마인드** (총 가치 요약)
- 🎯 **단일 & 반복 CTA 버튼** (크고 눈에 띄게)
- 💪 **마지막 동기부여 문구** (행동 촉구)

### 디자인 패턴
```
[대형 CTA 섹션]
  [배경: 그라디언트 또는 단색]
  [헤드라인: "지금 시작하세요"]
  [혜택 요약: 3줄 이내]
  [대형 버튼: px-12 py-5]
  [마이크로카피: "신용카드 불필요 • 언제든 해지 가능"]
```

## 1️⃣4️⃣ **푸터**

### 포함 요소
- 🏢 **회사/서비스 정보**
- 📧 **연락처 정보** (이메일, 전화번호)
- 📄 **법적 문구** (개인정보처리방침, 이용약관)
- 🔗 **SNS 링크** (인스타, 유튜브 등)
- ©️ **저작권 표시**

### 디자인 패턴
```
[3-4컬럼 푸터]
  컬럼 1: 로고 + 간단한 소개
  컬럼 2: 주요 링크 (제품, 가격, 회사)
  컬럼 3: 고객지원 (FAQ, 문의)
  컬럼 4: SNS 아이콘
[하단: 저작권 + 법적 링크]
```

### 기본 플로우 (14단계)
1. 헤더 & 히어로
2. 긴급성/신뢰 배지 (간단히)
3. 문제 공감
4. 문제 정의 + 해결책 제안
5. 솔루션 상세 설명
6. 사회적 증거 (숫자)
7. Before/After 대비

9. 권위 섹션 (미디어/전문가)
10. 사용성 강조
11. 혜택 종합 + 보너스
12. 타겟 필터링
13. 리스크 제거 (FAQ, 환불 정책)

3. 솔루션 설명

  </div>
</section>

## ⚠️ **주의사항**

### 반드시 포함할 요소
- ✅ 명확한 CTA (최소 3회 반복)
- ✅ 사회적 증거 (숫자, 후기)
- ✅ 리스크 제거 (환불, 보장)
- ✅ 긴급성 (한정, 마감 임박)

### 피해야 할 요소
- ❌ 불필요한 애니메이션 과다
- ❌ 너무 많은 정보 (한 섹션에 1가지 메시지)
- ❌ 읽기 어려운 폰트 크기
- ❌ 모바일 미대응 레이아웃

## 📱 **반응형 가이드**

### 모바일 (~ 768px)
- 컬럼을 1줄로 변경 (grid-cols-1)
- 텍스트 크기 축소 (text-4xl → text-3xl)
- 패딩 축소 (py-24 → py-16, px-8 → px-4)
- 버튼을 전체 너비로 (w-full)

### 태블릿 (768px ~ 1024px)
- 2컬럼 레이아웃 (grid-cols-2)
- 적절한 패딩 유지

### 데스크탑 (1024px ~)
- 3-4컬럼 레이아웃
- 최대 너비 제한 (max-w-6xl)
- 충분한 여백
